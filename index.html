What is the World Wide Web?

It is a collection of HTML documents!

What is HTML?

HTML stands for Hypertext Markup Language. It is the basis for almost every web page!

HTML is made up of:
- text content (what you see)
- mark up (how is it arranged)
- references to other documents (images and videos)
- hyperlinks (links between pages)

Some basic HTML vocabulary

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and closing tag. such as the paragraph tag. Some tags called "void" tags do not require a closing tag. An example of that is the break tag.

Element: An HTML element refers to everything within a set of opening and closing tags

Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Facebook URL, you would write Facebook.
